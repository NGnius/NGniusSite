--- layout: default+ title: Verify Game (Robocraft Launcher) ---

1. Open Robocraft Launcher
2. Wait for wrench to appear (the launcher usually does a quick update check everytime it opens, so that has to finish before the wrench will appear)
3. Select wrench icon
4. Select Ok
5. Wait for it to complete
6. Congratulations, you have verified the game! Now go test the game out to see if that fixed your problem.
