ModScraper

The program that currently doesn't do much, except find olds posts that have been bumped. This only works with the formating in the Robocraft forum sections at the moment. Sorry!


Github Repository
Current Features

necroDetector.py features

  • Detect a necro by running necroDetector.py
  • Choose what sections to monitor by adding or removing URLs in Resources/ForumsToCheck.txt
  • All of necroDetector.py's dependencies are found in /Resources (which is nothing but the ForumsToCheck.txt file right now)

    • scraper.py features

  • Retrieve pages with scraper.page.ret([url])
  • Raw HTML page storage in scraper.page.raw
  • HTML-less page (anything between a < and > are ignored) storage in scraper.page.text
  • Search strings with scraper.wordSearcher.wordSearcher([string to find], [string to search])
  • Search scraper.page.text with scraper.find([string to find])
  • All of scraper.py's dependencies are found in /scraperResources
