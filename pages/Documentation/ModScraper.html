ModScraper

The program that currently only detects profanity listed in Profanity.txt and bumps from a different year. The program checks the specified forum sections listed in ForumsToCheck.txt. This only works with the formating in the Robocraft forum sections at the moment. Sorry!


Github Repository
Current Features

General features

  • General purpose scraper.py modified for forum scraping
  • Configure general behaviour with /Resources/config.txt
  • Choose what forum sections to scrape with /Resources/ForumsToCheck.txt

    • capsDetector.py notable features

  • Detect all caps titles with capsDetector.isCapsTitle(post, caps threshold)
  • All of capsDetector.py's dependencies are found in /Resources

    • profanityDetector.py features

  • Detect profanity in posts with profanityDetector.isProfanity(post, swear threshold)
  • Choose which profanity will be detected and the weight of it with /Resources/Profanity.txt
  • All of profanityDetector.py's dependencies are found in /Resources

    • necroDetector.py features

  • Detect bumped threads with necroDetector.isNecro(post, time delta threshold)
  • All of necroDetector.py's dependencies are found in /Resources
