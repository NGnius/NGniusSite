ModScraper

The program that currently only detects profanity listed in Profanity.txt and bumps from a different year. The program checks the specified forum sections listed in ForumsToCheck.txt. This only works with the formating in the Robocraft forum sections at the moment. Sorry!


Github Repository
Current Features

capsDetector.py features (not yet fully implemented)

  • Detect ALL CAPS TITLES!!! by running capsDetector.capsTDetector()
  • Detect ERMMAGHEERRDH CAPS TITLES in the last post by running capsDetector.detectCapsT(url [,mode])
  • Choose what sections to monitor by adding or removing URLs in Resources/ForumsToCheck.txt
  • All of capsDetector.py's dependencies are found in /Resources

    • profanityDetector.py features

  • Detect profanity by running profanityDetector.profanityDetector()
  • Detect profanity in the last post by running profanityDetector.detectProfanity(url [,mode])
  • Choose what sections to monitor by adding or removing URLs in Resources/ForumsToCheck.txt
  • Choose what words to count as swear words by adding or removing words in Resources/Profanity.txt
  • All of profanityDetector.py's dependencies are found in /Resources

    • necroDetector.py features

  • Detect a necro by running necroDetector.necroDetector
  • Detect a necro in the last post by running necroDetector.detectNecro(url [,mode])
  • Choose what sections to monitor by adding or removing URLs in Resources/ForumsToCheck.txt
  • All of necroDetector.py's dependencies are found in /Resources (which is only ForumsToCheck.txt)

    • scraper.py features

  • Retrieve pages with scraper.page.ret(url)
  • Raw HTML page storage in scraper.page.raw
  • HTML-less page (anything between a < and > are ignored) storage in scraper.page.text
  • Search strings with scraper.wordSearcher.wordSearcher(string to find, string to search)
  • Search scraper.page.text with scraper.find(string to find)
  • All of scraper.py's dependencies are found in /scraperResources
